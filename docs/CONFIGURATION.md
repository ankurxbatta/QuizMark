# Configuration

All settings live in `.env`. Run `setup.sh` (Mac/Linux) or `setup.bat` (Windows) to create it automatically, or copy `.env.example` manually.

---

## Required Variables

| Variable | Description |
|---|---|
| `SECRET_KEY` | Signs JWT tokens. Auto-generated by setup script. Use `python3 -c "import secrets; print(secrets.token_hex(32))"` to generate manually. |
| `ADMIN_PASSWORD` | Instructor login password. Required. |
| `GEMINI_API_KEY` | Free-tier embeddings. Get at [aistudio.google.com](https://aistudio.google.com/app/apikey). |
| `OPENAI_API_KEY` | Primary LLM provider (vision, math, generation, marking). Get at [platform.openai.com](https://platform.openai.com/api-keys). |
| `ANTHROPIC_API_KEY` | Fallback LLM provider. Get at [console.anthropic.com](https://console.anthropic.com). |

## Auth

| Variable | Default | Description |
|---|---|---|
| `SECRET_KEY` | — | JWT signing secret. Required. |
| `JWT_ALGORITHM` | `HS256` | JWT algorithm. Do not change. |
| `JWT_EXPIRY_MINUTES` | `30` | Token lifetime in minutes. |
| `MAX_FAILED_LOGIN_ATTEMPTS` | `3` | Failed logins before account lockout. |
| `LOCKOUT_DURATION_MINUTES` | `5` | Lockout duration. |
| `ADMIN_ENABLED` | `true` | Creates admin account on first startup. |
| `ADMIN_USERNAME` | `admin` | Admin login username. |
| `ADMIN_PASSWORD` | — | Admin login password. Required. |
| `ADMIN_ROLE` | `instructor` | Role assigned to the admin account. |

## Gemini (Embeddings)

| Variable | Default | Description |
|---|---|---|
| `GEMINI_API_KEY` | — | Required. Used for 768-dim embeddings only. |
| `GEMINI_EMBEDDING_MODEL` | `gemini-embedding-001` | Do not change — MongoDB index is built on 768-dim output of this model. |

| Variable | Default | Description |
|---|---|---|

## LLM Routing

| Variable | Default | Description |
|---|---|---|
| `GENERATION_LLM_ENABLED` | `true` | Enable LLM question generation. |
| `GENERATION_LLM_PROVIDER` | `openai` | Primary generation provider. |
| `GENERATION_LLM_MODEL` | `gpt-4o-mini` | Generation model. |
| `GENERATION_MAX_TOKENS` | `4096` | Max tokens per generation call. |
| `ONLINE_LLM_ENABLED` | `true` | Enable LLM answer marking. |
| `ONLINE_LLM_PROVIDER` | `openai` | Primary marking provider. |
| `ONLINE_LLM_MODEL` | `gpt-4o-mini` | Marking model. |
| `LLM_TEMPERATURE` | `0.2` | Generation temperature. |
| `LLM_MAX_TOKENS` | `1024` | Max tokens per marking call. |

## MongoDB

|---|---|---|

## Celery / Redis

| Variable | Default | Description |
|---|---|---|
| `CELERY_BROKER_URL` | `redis://broker:6379/0` | Redis task queue URL. |
| `CELERY_RESULT_BACKEND` | `redis://broker:6379/0` | Redis result storage URL. |

---

## Marking Confidence Thresholds

| Variable | Default | Description |
|---|---|---|
| `CONFIDENCE_HIGH` | `0.85` | SLM score ≥ this → mark accepted without LLM call. |
| `CONFIDENCE_MID` | `0.55` | Score between MID and HIGH → RAG + OpenAI marking. Score below MID → wide RAG + OpenAI + flagged for review. |

---

## PDF Ingestion

| Variable | Default | Description |
|---|---|---|
| `UPLOAD_MAX_SIZE_MB` | `25` | Maximum PDF upload size. |
| `PDF_MAX_PAGES` | `700` | Maximum pages to process per PDF. |
| `PDF_MIN_CHUNK_CHARS` | `300` | Chunks shorter than this are discarded. |
| `PDF_MAX_CHUNK_CHARS` | `3000` | Chunks longer than this are split. |
| `INGEST_PAGE_WINDOW` | `6` | Pages processed per checkpoint window. |
| `INGEST_TIME_BUDGET_SECONDS` | `1500` | Worker re-queues the task after this long (stays under Celery 30-min soft limit). |
| `ENABLE_VISION_EXTRACTION` | `true` | Enable chart/image vision descriptions. |

---

## RAG Retrieval

| Variable | Default | Description |
|---|---|---|
| `TOP_K_RETRIEVAL` | `5` | Chunks retrieved per query in standard RAG. |
| `TOP_K_WIDE_RETRIEVAL` | `10` | Chunks retrieved when confidence is low (wider context). |
| `SIMILARITY_THRESHOLD` | `0.75` | Minimum cosine similarity for a chunk to be included in results. |

---

## Question Generation

| Variable | Default | Description |
|---|---|---|
| `GEN_CHAPTER_CONCURRENCY` | `5` | Chapters generating questions in parallel. |
| `DEDUP_SIMILARITY_THRESHOLD` | `0.92` | Embedding cosine ≥ this → question is a near-duplicate and dropped. |

---

## CORS

| Variable | Default | Description |
|---|---|---|
| `CORS_ORIGINS` | `http://localhost:3000` | Comma-separated list of allowed frontend origins. Add your production domain here. |
